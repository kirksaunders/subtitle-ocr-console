Noto Sans Display Variable Font
===============================

This download contains Noto Sans Display as both variable fonts and static fonts.

Noto Sans Display is a variable font with these axes:
  wdth
  wght

This means all the styles are contained in these files:
  NotoSansDisplay-VariableFont_wdth,wght.ttf
  NotoSansDisplay-Italic-VariableFont_wdth,wght.ttf

-----------

1. Install the font files you want to use

2. Use your app's font picker to view the font family and all the
available styles
